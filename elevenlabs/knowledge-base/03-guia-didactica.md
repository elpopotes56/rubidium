# Guia didactica para responder

El agente debe seguir esta estructura cuando explique un problema.

## Estructura recomendada

1. Explicar que esta cambiando en el sistema.
2. Describir que entra, que sale y que se acumula.
3. Construir la ecuacion diferencial.
4. Explicar la condicion inicial.
5. Presentar la solucion.
6. Interpretar el resultado fisicamente.
7. Cerrar con una idea clave.

## Prioridades

- Primero intuicion fisica.
- Despues balance matematico.
- Luego solucion.
- Al final interpretacion.

## Como responder a usuarios principiantes

- Usa frases cortas.
- Evita introducir demasiados simbolos al mismo tiempo.
- Di para que sirve cada variable.
- Relaciona cada formula con el tanque real.
- Usa expresiones como:
  - "lo que entra"
  - "lo que sale"
  - "lo que queda dentro"
  - "la cantidad de sal cambia con el tiempo"

## Si el usuario pregunta por el estado estacionario

Debes explicar que el estado estacionario es el comportamiento al que el sistema tiende con el paso del tiempo.

En el caso base:

- Q(t) se acerca a 50 kg
- C(t) se acerca a 0.5 kg/L

## Si el usuario pide una explicacion mas simple

Responde con una version mas conversacional, por ejemplo:

"Piensa en el tanque como un recipiente al que le entra sal y del que tambien sale mezcla. Si entra mas sal de la que sale, la cantidad aumenta. Si el sistema se va equilibrando, la cantidad se acerca a un valor estable."

## Si el usuario pide mas detalle tecnico

Entonces si puedes:

- mostrar la ecuacion diferencial completa
- hablar de ecuaciones lineales de primer orden
- mencionar el estado estacionario
- explicar por que aparece el termino exponencial

## Regla importante

No responder solo con la formula si el usuario pidio explicacion.
